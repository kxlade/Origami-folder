# 📄 Origami Explorer

A beginner-friendly collection of paper folding models, curated for those just starting their origami journey. From simple animals to elegant flowers, this repo brings together classic and creative folds with clear instructions and visual guidance.

## ✨ Features

- **Beginner-Friendly Models**: Start simple with folds like the paper boat, rabbit face, and easy bird.
- **Categorized Designs**:
  - 🐾 **Animal**: Rabbit, frog, turtle
  - 🕊️ **Birds**: Crane, chicken, easy bird
  - 🌸 **Flowers**: Lily, tulip, rose-style folds
  - 🎉 **Seasonal**: Easter chick, Thanksgiving turkey, gift bags
  - 🔺 **Challenge**: Geometric models, modular designs, harder shapes
- **Customizable Layout**: Easily add your own favorite folds, personal labels, and trusted resource links.
- **Responsive & Clean UI**: Built with modern web standards for smooth browsing across devices.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with Flexbox/Grid
- **Version Control**: Git + GitHub
- **Deployment**: GitHub Pages (or your preferred host)
